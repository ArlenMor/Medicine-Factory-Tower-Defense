# Tower Defence

Гибрид tower defence, фермерства и крафта. Защищайте базу от волн врагов, выращивайте ресурсы, выполняйте заказы и улучшайте свои возможности. 10 уровней с возрастающей сложностью.

## Основные системы

**Tower Defence** — Расстановка турелей и баррикад на сетке 6x5. Турели автоматически атакуют врагов в радиусе поражения, баррикады блокируют путь и принимают урон на себя.

**Враги** — Три типа: Scout (быстрый, слабый), Gnawer (средний), Tank (медленный, прочный). Атакуют баррикады, затем — центральную базу (Мозг). Поражение наступает при уничтожении Мозга.

**Волны** — Волны врагов с настраиваемым составом. Если последняя волна не уничтожена вовремя — зацикливается, создавая постоянное давление.

**Фермерство** — Грядки с шестью слотами. Три вида растений (Crystal, Polymer, NanoGel) проходят стадии роста и приносят ресурсы при сборе.

**Крафт и заказы** — Станция крафта, на которой нужно выполнять заказы, тратя собранные ресурсы на производство лекарств. Выполнение заказов приносит кредиты и продвигает уровень к победе.

**Улучшения** — Три ветки: скорость роста растений, шанс двойного сбора (с гарантией после нескольких промахов), скорость крафта.

**Клик по врагам** — Дополнительный урон по врагам при клике мышью с кулдауном.

**Обучение** — Пошаговые туториалы для первых уровней с подсветкой элементов интерфейса.

**Локализация** — Полная поддержка английского и русского языков.

## Технологии

Unity 6 (6000.3.8f1), Universal Render Pipeline, C#, DOTween, Odin Inspector, R3, Unity Input System, TextMesh Pro.

## Начало игры

Откройте сцену `Assets/_Project/Content/Scenes/` и запустите проект. Точка входа — `Bootstrap`.
